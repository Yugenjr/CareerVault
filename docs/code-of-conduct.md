# Code of Conduct

This project is intended to be welcoming, respectful, and practical for everyone who uses or contributes to it.

## Expected Behavior

- Be respectful in discussion and review feedback.
- Focus criticism on ideas and code, not people.
- Assume good intent when collaborating.
- Keep communication clear and relevant.

## Unacceptable Behavior

- Harassment or discrimination of any kind
- Personal attacks or abusive language
- Deliberate disruption of discussions or reviews
- Sharing private information without permission

## Reporting Issues

If you encounter behavior that does not fit these expectations, raise it through the project maintainers or the usual repository communication channel.

## Enforcement

Maintainers may remove comments, pause reviews, or take other reasonable actions when conduct is inconsistent with this code of conduct.
